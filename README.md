# Инструкция по использованию

1. Скачать с [сайта](https://www.python.org) последнюю версию IDLE Python;

2. Скачать данный репозиторий нажав на кнопку [Clone or download](https://github.com/zakolm/parse_xlsx/archive/master.zip);

3. Скачать все необходимые библиотеки(используя команды ниже в командной строке):

```bash
python -m pip install pyexcel --user
```

```bash
python -m pip install smtplib --user
```

4. Изменить данные в файле **config.ini**, внести свои данные;

5. Добавить новый/изменить прилогающийся xlsx-файл;

6. Запустить скрипт с помощью bat-файла(просто кликнув на **click.bat**).
